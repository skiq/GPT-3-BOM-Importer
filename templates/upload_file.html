The Problem with BOMs

BOM files usually contain MPNs (Manufacturer Part Number). But BOMs are not standardized so every Excel sheet looks completely different.
Sometimes the MPNs can't even be recognized at first sight because they might be listed in the comments column.
We created a tool where you can upload any BOM file and get back a CSV with an additional column displaying validated MPNs.
The validation is done with the Octopart API.
